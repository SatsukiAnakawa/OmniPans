 OmniPans v1.2.0
==================================================

Thank you for downloading OmniPans.

This is a utility app that allows you to easily adjust the volume and pan 
(left/right balance) of all your audio devices on Windows from a single location.

[ How to Use ]
1. Run "OmniPans.exe" located in this folder.
2. An icon will appear in the system tray.
   - Left-click the icon to open the device list window.
     - In the window, click the speaker icon for any device to toggle mute.
     - Click the pan (balance) icon to reset the balance to the center or restore its previous value.
   - Right-click the icon to open the settings menu.

[ System Requirements ]
- Windows 10 / 11
- .NET 8.0 Desktop Runtime
  (If the application does not run, please download and install it from Microsoft's official website)

【このソフトについて】
Windowsの全オーディオデバイスの音量とパン（左右バランス）を、
一つの場所から手軽に調整できるユーティリティアプリです。

【使い方】
1. このフォルダにある「OmniPans.exe」を実行してください。
2. タスクトレイにアイコンが表示されます。
   - アイコンを左クリック: デバイス一覧ウィンドウを開きます。
     - 各デバイスのスピーカーアイコンをクリックするとミュートのON/OFFが切り替えられます。
     - パン（左右バランス）のアイコンをクリックすると、バランスを中央にリセットしたり、元の値に戻したりできます。
   - アイコンを右クリック: 設定メニューを開きます。

【動作環境】
- Windows 10 / 11
- .NET 8.0 Desktop Runtime
  (もし動作しない場合は、Microsoftの公式サイトからダウンロード・インストールしてください)
